空间计算 / 效率工具

核心理念“好好吃饭 = 好好生活”

针对18-35岁年轻用户健康饮食痛点，设计了这款基于空间计算的沉浸式电子日历。借鉴 VisionPro 交互优势，填补传统日历功能单一的空白。

设计细节

采用毛玻璃质感 (Glassmorphism) 以适应复杂的空间背景。通过 3D 蔬菜的生长动画反馈时间的流逝，长按手势即可记录饮食心情。
